# Module de tests

## Raison d’être

Ce module contient les tests unitaires permettant de vérifier
le bon fonctionnement de l’application.

---

## Couverture des tests

Les tests vérifient :

- Les opérations arithmétiques (addition, soustraction, multiplication, division)
- La gestion des erreurs
- Le traitement des expressions invalides

---

## Exécution des tests

Depuis la racine du projet :

```bash
pytest
```
ou
```bash
python -m pytest
```
