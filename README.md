# Calculatrice Web

## Numéro d’équipe
Équipe 66

## Objectif
Développer une application web de calculatrice simple à l’aide de Flask en Python.  

## Prérequis d’installation
- Python 3.11.9
- pip
- Git
- Un navigateur web

## Instructions d’installation

1. Cloner le dépôt :
   ```bash
   git clone https://github.com/Simonbri/TP3-LOG3000-Equipe66.git

2. Installer les dépendances :
   ```bash
   cd TP3-LOG3000-Equipe66
   pip install -r requirements.txt

3. Lancer l'application :
   ```bash
   python app.py

4. Ouvrir dans un navigateur :
   ```cpp
   http://127.0.0.1:5000

## Organisation modulaire

Le projet est divisé en trois modules principaux :

- Backend : logique et traitement des calculs
- Templates : interface HTML
- Static : ressources CSS
